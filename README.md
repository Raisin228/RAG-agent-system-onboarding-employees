# RAG-agent-system-onboarding-employees
Система для автоматизации онбординга новых сотрудников на основе RAG-архитектуры с мультимодальным вводом.


«Разработка интеллектуального чат-бота для автоматизации онбординга сотрудников на основе RAG-архитектуры с мультимодальным вводом.»


Бизнес-требование:
Cотрудник приходит в компанию и вместо того, чтобы дёргать коллег, пишет боту в Telegram. Бот умеет принять текстовый или голосовой вопрос, найти ответ в корпоративной базе знаний и вернуть его со ссылкой на источник. Стриминг ответа, чтобы не было ощущения зависания.

Конечный результат:
1. Работающий Tg-бот c демо на синтетическом датасете [настоящие данные мне никто не даёт]. Я их нагенерю сам очень качественно.
2. RAG-пайплайн: ingestion -> chunking -> embeddings -> Qdrant -> retrieval -> LLM-генерация
3. Классификатор запросов. Поиск ответов по векторной базе знаний.
4. Voice-to-Text через Whisper. Т.е в систему можно будет не просто написать вопрос текстом, но и просто кинуть голосовуху, по которой она сама разберётся что тебе нужно)
5. Сравню разные стратегии чанкирования для векторных БД. Ведь одна из основных задач: а как правильно снабжать модель контекстом?
