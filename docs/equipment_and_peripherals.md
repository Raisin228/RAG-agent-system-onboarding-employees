## Как получить оборудование в первый день

Оборудование подготавливает IT-отдел. Сергей Новожилов (@sergey.novozhilov) выдаёт его в офисе (стойка IT, 3-й этаж, комната 304) или отправляет курьером если работаешь удалённо.

Курьерская доставка: не позже чем за 3 рабочих дня до выхода HR согласовывает с IT адрес доставки. Если что-то не пришло к первому дню — пиши в **#it-support**.

## Дополнительное оборудование

### Второй монитор

Запрос в **#it-support**: «Прошу второй монитор для рабочего места». Одобряется автоматически для разработчиков и QA. Срок: 3–5 рабочих дней.

### Эргономика (кресло, подставка, внешняя клавиатура)

Бюджет: **10 000 ₽** на эргономику при выходе (разово). Можно потратить на:
- Эргономичное кресло (если работаешь из дома)
- Подставку для ноутбука
- Альтернативную клавиатуру / мышь
- Коврик для мыши с подставкой для запястья

Оформляется как расходы через форму в Confluence (см. `expense_reimbursement.md`).

### Замена оборудования

Если оборудование сломалось — пишешь в **#it-support**. Стандартные сроки:
- Замена аксессуара (мышь, клавиатура, кабель): 1–2 рабочих дня.
- Замена ноутбука: до 3 рабочих дней (выдаётся временный пока чинят/меняют основной).

Гарантийный ремонт — IT организует самостоятельно, тебе ничего делать не нужно.

## Корпоративный телефон

Корпоративный телефон выдаётся сотрудникам, у которых есть обязанности on-call или которые ведут переговоры с клиентами (аккаунт-менеджеры, продакты). Разработчикам — только если в регулярной on-call ротации.

Модели:
- iPhone 15 128 GB (основная выдача)
- Samsung Galaxy S24 (по запросу)

Тариф: корпоративная SIM Тинькофф Мобайл, безлимитные звонки по России, 50 GB интернет. Зарубежный роуминг — по заявке в **#it-support** не позже чем за 2 дня до поездки.

## Программное обеспечение

Все лицензии ПО хранятся в IT-отделе. Стандартный набор устанавливается автоматически через Jamf Pro (MDM) на корпоративный MacBook при первом входе. Включает:

| ПО | Назначение |
|---|---|
| CrowdStrike Falcon | Антивирус (обязательно, см. `security_policy.md`) |
| 1Password | Менеджер паролей |
| Slack | Мессенджер |
| Zoom | Видеозвонки |
| JetBrains Toolbox | IDE (GoLand, DataGrip, WebStorm) |
| VS Code | Редактор (опционально) |
| TablePlus | GUI для PostgreSQL и Redis |
| Docker Desktop | Контейнеры локально |

Если нужно ПО не из этого списка — запрос в **#it-support** с обоснованием. IT проверяет лицензию и соответствие политике безопасности.

### JetBrains лицензии

У FinBridge корпоративная лицензия на весь пакет JetBrains (All Products Pack). Активация:
1. Открой любую IDE.
2. Выбери «Log in to JetBrains Account».
3. Войди через `имя.фамилия@finbridge.io`.
4. Лицензия подтянется автоматически.

## Возврат оборудования

При увольнении всё оборудование возвращается в IT-отдел (Сергей Новожилов) в последний рабочий день. Не возвращённое оборудование вычитается из финального расчёта по рыночной стоимости б/у устройства.
